CeGCC 0.59.1 uses gcc 4.4.0
The latest official release (0.55) is based on gcc 4.1.0.

The list of software contained in cegcc :

  • gcc
    • CeGCC > 0.59 : use gcc 4.4.0
    • previous versions : gcc 4.1.0
  • binutils : CVS snapshot taken on 4 February 2009, this is around the binutils 2.19.1 timeframe
  • newlib : 1.14.0 (released December 2005)
  • mingw : CVS snapshot taken on 6 February 2009
  • w32api : CVS snapshot taken on 6 February 2009
  • gdb
    • gdb-6.8 works well for ARM.
    • For Windows CE Embedded on x86, a number of issues need to be resolved. Work is progressing on this, but this is not in a released version of gdb yet.

Note that arm-cegcc contains newlib, the arm-mingw32ce and i386-mingw32ce packages do not, they contain a mingw-based library instead.
