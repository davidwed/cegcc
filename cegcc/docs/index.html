Required software

Use of the cross-debugger between a CE system and a Linux host PC requires a working installation of SynCE. SynCE takes care of hooking up the PDA via TCP/IP, implements the RAPI library, etc. On Windows, this functionality is part of ActiveSync which usually ships with your PDA.
